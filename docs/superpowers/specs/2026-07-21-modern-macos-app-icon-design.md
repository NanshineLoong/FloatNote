# FloatNote 现代 macOS 应用图标设计

## 目标

修复当前应用图标在 Launchpad、Finder 和 Dock 中出现的白色外围轮廓，同时保留 FloatNote 已有的蓝灰背景、斜放纸张和花括号识别元素。新图标采用已确认的“C · 现代 macOS”方向，不扩展为品牌重设计。

## 视觉设计

- 使用 1024×1024 正方形母版。
- 蓝灰色径向或柔和渐变背景铺满整个画布，保持完全不透明；不在母版中预先绘制外层系统圆角，也不留下外围白色画布。
- 纸张约占画布宽度的 74%，保持轻微逆时针旋转，位置视觉居中。
- 纸张圆角相较当前素材略小，避免与系统外层圆角形成重复轮廓。
- 保留暖白纸张、蓝灰花括号与两条书写线；小尺寸下以花括号和纸张轮廓作为主要识别特征。
- 阴影改为更短、更柔和、透明度更低的环境阴影，避免当前图标边缘的深色浑浊感；不在画布外沿烘焙白色或深色光晕。
- 由 macOS 对正方形资源施加最终圆角蒙版。

## 资源与生成方式

- 在 `src-tauri/icons/` 中保留一个可重复编辑的高分辨率源文件，作为应用图标的唯一视觉源。
- 通过脚本或 Tauri 图标生成工具，从该源文件重新生成现有的 PNG、ICNS、ICO 和 Windows Store 尺寸，避免各平台资源漂移。
- 不修改 `tauri.conf.json` 的图标引用方式；DMG 中的 `FloatNote.app` 继续使用 `Contents/Resources/icon.icns`。
- 托盘图标不在此次范围内，避免把彩色应用图标的视觉规则误用到菜单栏模板图标。

## 验证

- 验证母版为 1024×1024，背景边缘完全不透明且不存在近白色外围画布。
- 解包 `icon.icns`，确认包含 16、32、128、256、512 点的 1x/2x 资源，最大档为 1024×1024。
- 检查生成的各平台图标来自同一母版，并确认 Tauri 配置引用的文件均存在。
- 重新构建 DMG，挂载后确认 `.app/Contents/Resources/icon.icns` 与仓库生成文件一致。
- 在 Finder、Dock 和 Launchpad 中目视检查：无白色外框、主体未被系统圆角裁切、16px 和 32px 下花括号仍可辨认。

## 非目标

- 不更改应用名称、Bundle ID、签名或 DMG 安装界面。
- 不引入 dark、tinted 或 Liquid Glass 多外观变体；本次先交付兼容当前 Tauri 打包链的扁平 ICNS。
- 不修改应用内功能图标或托盘图标。
