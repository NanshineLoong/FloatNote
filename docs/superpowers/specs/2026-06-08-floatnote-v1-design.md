# FloatNote v1 设计文档

- 日期：2026-06-08
- 状态：已确认，待实现

## 1. 产品定位

FloatNote v1 是一个**始终置顶的悬浮 Markdown 速记窗口**，风格贴近 macOS 原生应用。核心价值：在阅读（或任意应用）时，用全局快捷键把划线原文以引用格式低摩擦地收进当前笔记，不打断心流；笔记以 Markdown 文件存到用户指定的目录。

v1 刻意**不做**：浏览器页面上下文绑定（URL/标题）、语音转文字、收件箱/瞬间分类、知识图谱、多 Agent。这些来自 Garden 产品文档的后续演进方向，不属于本版本。

成功标准：阅读中捕捉的注意力成本显著低于「切到笔记软件粘贴」。

## 2. 技术栈

- **Tauri v2**：桌面应用。
- **Rust 后端**：托盘、全局快捷键、窗口置顶/显隐、划线文字抓取、Markdown 文件读写、配置持久化。
- **Web 前端**：**Vanilla TypeScript + Vite，不用 UI 框架**。Markdown 编辑器（CodeMirror 6）+ 原生风格 UI。图标用 **Phosphor**（SVG，regular 权重贴近 SF Symbols，不用 emoji）。
- 前后端通过 Tauri command（前端调后端）和 event（后端通知前端，如「已追加引用」）通信。

主要依赖：`tauri-plugin-global-shortcut`、剪贴板能力（Tauri clipboard）、文件读写（Rust std 或 `tauri-plugin-fs`）、托盘（Tauri tray）。前端：CodeMirror 6（markdown 语言包 + 自定义主题）、Phosphor 图标。

**前端框架决策（Vanilla TS）**：v1 交互面很小——重活由 CodeMirror 扛（框架无关，直接挂 DOM），其余响应式状态仅「当前目录 + 当前笔记 + 笔记列表」几项，手写管理足够。不用框架可保持包小、构建快、依赖少。若 v2 引入收件箱/分类等复杂 UI，再考虑迁移到 Svelte。用 Tauri 的 `vanilla-ts` 脚手架模板。

平台：v1 仅针对 macOS。

## 3. 应用形态：托盘应用 + 两个窗口

- **不进 Dock**：accessory 应用（macOS `LSUIElement` / Tauri `ActivationPolicy::Accessory`），只在菜单栏有一个托盘图标。
- **托盘交互**：
  - 左键点击托盘图标 → 切换悬浮笔记窗口显隐。
  - 托盘菜单（右键）：`显示/隐藏笔记`、`设置…`、`退出`。
- **两个窗口，按需创建**：
  1. **悬浮笔记窗口**：常驻置顶，核心界面。
  2. **设置窗口**：独立普通窗口，仅从托盘菜单打开。笔记窗口**没有设置入口**。

## 4. 核心交互：两个全局快捷键

| 功能 | 默认键（设置窗口可改） | 行为 |
|---|---|---|
| 划线引用 | `⌥⌘C` | 抓取当前划线文字 → 格式化为引用块 → **静默追加**到当前笔记末尾 → **不抢焦点**。窗口隐藏时也照常写入。 |
| 显示/隐藏笔记窗口 | `⌥⌘N` | 切换悬浮笔记窗口显隐（与左键点托盘等效）。窗口可见时常驻置顶。 |

### 划线抓取机制（模拟 Cmd+C + 读剪贴板）

按下「划线引用」时：

1. 备份当前系统剪贴板内容。
2. 模拟发送 `Cmd+C` 给前台应用。
3. 短暂等待（如 ~80ms）后读取剪贴板。
4. 若剪贴板**有新文字** → 进入引用格式化与追加流程。
5. 若剪贴板**无变化 / 为空**（没有选中）→ 静默忽略，不写入笔记。
6. 还原步骤 1 备份的剪贴板。

需要 macOS 辅助功能（Accessibility）权限来模拟按键。首次触发若未授权，引导用户去「系统设置 → 隐私与安全性 → 辅助功能」开启。

### 引用格式（纯引用块）

- 单行原文：`> 原文`
- 多行原文：每行加 `> ` 前缀。
- 追加时在前后留一个空行，与已有内容隔开。
- 不附带来源应用名、不附带时间戳。

## 5. 悬浮笔记窗口

### 顶部极简栏（从左到右）

```
[目录名 ▾]  /  [笔记名 ▾]                    [ + 新建 ]
```

- **目录名**：显示当前工作目录的名称（basename，全路径作 hover 提示）。点击 → 弹出 **macOS 系统目录选择窗口**；选定后切换当前工作目录，笔记列表与新建都基于该目录。
- **斜杠分隔** `/`。
- **笔记名 + 下箭头（Phosphor caret-down）**：显示当前笔记文件名（去 `.md` 扩展名）。点下箭头 → 展开一个小 menu，列出**当前目录下其他 `.md` 文件**；点击某项即在下方主区域载入该笔记。
- **+ 新建（Phosphor plus）**：在当前目录新建一篇笔记并切为当前笔记。
- 顶栏**无设置入口**（设置在托盘菜单）。

### 主区域

- CodeMirror 6 编辑当前笔记的 Markdown 源码，**实时自动保存**（防抖 ~500ms 写盘）。
- 原生风格：干净留白、舒适字体与行距、轻量 Markdown 语法高亮（标题/引用块/列表/加粗等淡淡区分），跟随系统深浅色。
- 窗口小尺寸（默认约 380×520），可调整并记住上次尺寸/位置。

## 6. 笔记与文件模型

- 任意时刻存在唯一「**当前笔记**」= 当前目录下的一个 `.md` 文件，编辑即自动保存。
- **当前工作目录**：由顶栏目录选择器决定。应用记住**上次使用的目录**，启动时打开它；首次启动默认 `~/FloatNote/`（自动创建）。
- **文件名自动生成，不由标题派生**：新建时以**时间戳**命名，如 `2026-06-08 14-30.md`（同分钟内冲突则追加序号）。笔记**可以没有标题/无内容**。
- 顶栏「笔记名」= 文件名去扩展名。
- **可选重命名**：点击笔记名可直接重命名文件（非必须；用户也可一直用自动名）。重命名目标已存在则提示，不覆盖。
- 笔记列表 = 扫描当前目录下的 `.md` 文件（按修改时间倒序）。

## 7. 设置窗口

独立窗口，从托盘菜单打开，内容：

- 两个全局快捷键的绑定。
- 字体大小。
- （可选）开机自启动（launch at login）。

配置持久化到 Tauri app config 目录下的一个 JSON 文件。

## 8. 错误处理

- **抓不到划线文字**：静默忽略，还原剪贴板，不污染笔记。
- **辅助功能权限缺失**：首次触发时检测并引导去系统设置；未授权时划线引用降级（提示）。
- **文件写入失败**（目录不存在/无权限/磁盘满）：前端明确提示，编辑器内未保存内容**绝不丢弃**，可重试。
- **重命名冲突**（目标文件名已存在）：提示用户，不覆盖。
- **目录无 `.md` 文件**：笔记列表显示空态，可直接新建。
- **快捷键注册冲突**（被其他应用占用）：设置窗口提示注册失败，允许改键。

## 9. 测试策略

- **Rust 单元测试**：
  - 引用块格式化（单行 / 多行 / 含空行 / 含已有 `>` 的原文）。
  - 时间戳文件名生成与同分钟冲突序号。
  - 目录扫描排序、配置读写（默认值、缺字段兼容）。
- **前端逻辑测试**：
  - 自动保存防抖触发。
  - 目录切换 / 笔记切换时正确加载与落盘。
- **手动验证清单**（系统集成）：
  - 在 Safari / Preview / VS Code 划线后按快捷键，引用正确进入当前笔记且剪贴板被还原。
  - 窗口隐藏时划线引用仍写入。
  - 左键点托盘、全局快捷键均能切换窗口显隐。
  - 不出现在 Dock；置顶在全屏应用前的表现。
  - 目录选择器切换目录后列表与新建均落在该目录。
  - 辅助功能权限未授予时的引导流程。

## 10. 项目结构

```
FloatNote/
├── src-tauri/              # Rust 后端
│   ├── src/
│   │   ├── main.rs
│   │   ├── tray.rs         # 托盘图标、菜单、点击切换窗口
│   │   ├── windows.rs      # 笔记窗口 / 设置窗口的创建与显隐、置顶
│   │   ├── shortcuts.rs    # 全局快捷键注册与分发
│   │   ├── capture.rs      # 模拟 Cmd+C + 剪贴板抓取 + 还原
│   │   ├── quote.rs        # 引用块格式化（纯逻辑，可测）
│   │   ├── notes.rs        # 笔记文件读写、时间戳命名、目录扫描、重命名
│   │   └── config.rs       # 配置读写
│   └── tauri.conf.json     # accessory policy、两个窗口定义
├── src/                    # Web 前端（Vanilla TS + Vite）
│   ├── note/              # 笔记窗口：顶栏、目录/笔记切换、编辑器、自动保存
│   │   └── editor/        # CodeMirror 集成 + 原生主题
│   └── settings/          # 设置窗口
└── docs/superpowers/specs/
```

## 11. 明确不在 v1 范围内

浏览器 URL/标题绑定、语音转文字、阅读瞬间收件箱与分类、跨平台（Win/Linux）、多笔记同屏、富文本 WYSIWYG、知识图谱/AI 整理。
